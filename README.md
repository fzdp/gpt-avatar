## Features

- Realistic 3D avatar rendering
- Real-time conversations with ChatGPT-powered AI
- Smooth React Native user interface
- Robust Express backend support

## Tech Stack

- Mobile Application: React Native
- Backend: Express
- AI Conversation: ChatGPT API
